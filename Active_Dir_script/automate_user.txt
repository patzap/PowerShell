Powershell create AD users script

*Goal -> automate user account script, each parameter indicates the number of users to create, Name of user, and which OU it will go


Parameters when executing script
	-num or users
	-batch name of users
	-OUlocation



each parameter will loop and create a new AD user account until the condition equals the number of users specified in the parameter 


PS commands used in script
